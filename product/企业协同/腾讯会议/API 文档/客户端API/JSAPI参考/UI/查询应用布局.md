## 接口描述
- **描述：**ui.getWebAppPosition。
- **支持的版本：**3.9.0
- **是否需要鉴权：**否


## 输出参数
WebAppPositionInfoResp：

| 参数名称 | 参数类型 | 参数描述 |
| --- | --- | --- |
| position | Number | 应用位置：<br>1：弹窗<br>2：侧边栏<br>4：主视图内嵌<br>8：自定义布局<br>16：覆盖布局 |
| supportPosition | Number | 1：弹窗<br>2：侧边栏<br>4：主视图内嵌<br>8：自定义布局<br>16：覆盖布局 |
