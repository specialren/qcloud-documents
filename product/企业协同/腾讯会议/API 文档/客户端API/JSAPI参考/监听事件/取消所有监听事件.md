## 事件描述
**事件名：**removeAllEventListener
**事件说明：**取消所有监听事件。



## 示例
```plaintext
* 场景 1: 使用 import 引入 
import { event } from '@tencent/wemeet-js-sdk'; event.removeAllEventListener();  

场景 2:使用 script 引入 
<script src="xxxx""></script> wemeet.event.removeAllEventListener();  

or  

wemeet.removeAllEventListener();
```
